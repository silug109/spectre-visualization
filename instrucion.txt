Как работать?
легко

1) pip install -r requirments.txt
должно установить зависимости.
если чо, то добавляем pip installом

2)python main.py [ФЛАГИ СЮДА]

-data  - указываем директорию, где лежат .mat файлы. Без этого работать не будет
-filter - можно фильтровать файлы, которые лежат в директории. Пример: нужны файлы со словом Artyom, в regular expression стиле пишешь *Atyom*.mat
-output - опционально, если ничего не будет, то ничего не сохраняется(ни картинки, ни видосики) - режим визуализации. Если есть директория, то туда сохраняются картинки и потом еще из этих картинок сохраняется видосик, гифочка
-type - avi/gif

можно добавить много какого функционала:
визуализация из потока, сохранение файлов облаков точек в быстрые форматы, незалупленное проигрывание, а просто один проход, разрешение выходного видео/гифок, добавление подложки, линий всех там, работу не в автоматическом режиме,выставление порога фильтрации точек, короче просто скажите чаво еще сделать..

если в корне лежит файл viewpoint.json, то окно визуализации сразу ставит камеру на позицию, которая в файле содержалась.

Когда появляется окно визуализации, то можно его остановить можно закрытием на крестик на окошке, программа продолжит работу.
